# Large File Upload UI (React)

## Overview
A frontend application that enables users to upload large files using chunk-based upload with real-time progress tracking.

## Features
- File selection
- Chunk-based upload handling
- Upload progress indication
- Interaction with backend upload service

## Tech Stack
- React
- JavaScript 
- REST API integration

## How It Works
1. User selects a file
2. File is split into chunks in the browser
3. Chunks are sent sequentially or in parallel to the server
4. Progress is updated during upload

## Focus
Demonstrates client-side handling of large files and interaction with backend services.

## Note
UI is kept simple to focus on functionality and upload workflow.
